stages

Give a media file and a player, the status can be



stage

program

file

-1

unknown

unknown

1

not running

not running => not connected => not active

2

running, connection unknown


3

running & connected

unknown whether file is connected

4

running & connected

file NOT connected

5

running & connected

file connected, but not active

6

running & connected

connected and active



How to move from stage to stage:

  • 1 → 2

    • startProcess

  • 2 → 3

    • connectToProcess

  • 3 → 4 / 6

    • activateFileIfConnected

  • 4 → 6

    • activateNonConnectedFile
